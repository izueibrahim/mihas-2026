# GlobalOrganisersBar Component

## Overview
`GlobalOrganisersBar.jsx` is the small, ultra-thin banner located at the very apex of the website header. It identifies the principal entities backing the event: MITI, MATRADE, etc.

## Layout Logic
- Rendered using distinct colored badges and micro-text (`text-[10px]`).
- On small screens, the logos or lengthy text may collapse, re-centering items.
- It is visually isolated from main navigation context with varying shades of dark gray (`bg-slate-900`/`bg-slate-950`).
