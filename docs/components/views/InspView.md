# InspView Component

## Overview
`InspView.jsx` operates out of primary view bounds, detailing the heavily specialized International Sourcing Programme (INSP). This area is completely focused on generating high-ticket B2B matching and leads.

## Sections
1. **Overview**: Injects statistical analysis supporting the INSP capability (historic sales data).
2. **Key Sectors**: Highlights the industry focal points using horizontal scroll arrays or grids.
3. **Registration Tiers**: Explains the difference between Foreign Buyers and Local Sellers registration mechanisms conditionally via high-fidelity card layouts.

## Interaction
All primary CTAs loop directly back to `onOpenForm('visitor')` or similar data-gathering triggers to maximize registration volumes while users are assessing value.
