# Visitor Data (`visitor.js`)

## Overview
Manages the persuasive informational metrics targeting trade buyers and public users to attend the event.

## Structure
- `visitorProfileData`: Demographic definitions representing historical attendee archetypes (e.g., "Wholesalers", "Retailers", "E-Commerce").
- `whyVisitData`: Value-proposition cards highlighting product sourcing, B2B matching, and learning opportunities.
- `registrationStepsData`: An array outlining the sequential actions a user must take to acquire an access badge (Profile setup -> Verify -> Collect Badge).

## Component Integration
Provides the backbone for `VisitorView.jsx` tab content.
