# Stats Data (`stats.js`)

## Overview
Houses crucial historical figures utilized continuously as social proof and validation.

## Structure
Exports `statsData`, an incredibly simple 4-object array containing `value` and `label` pairs. 
- Example: `{ value: "RM4.3B", label: "Trade Value Generated" }`.

## Component Integration
Utilized immediately upon user arrival in `HomeView.jsx` building the global impact section.
