# Sectors Data (`sectors.js`)

## Overview
A lightweight data module detailing the overarching commercial sectors relevant to the Halal ecosystem constraint scope.

## Structure
Exports `sectors`, a hard-coded array of 14 key global industries verified by MATRADE (e.g., "Food and Beverage", "Pharmaceuticals", "Islamic Finance", "Muslim-Friendly Tourism").

## Component Integration
Utilized in drop-down menus within the `MultiStepForm` and `DirectoryView` to establish rigid filter taxonomy.
