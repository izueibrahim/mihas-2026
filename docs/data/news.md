# News Data (`news.js`)

## Overview
A highly structured array representing official Press Releases issued directly by MATRADE or event affiliates.

## Structure
Exports `newsData`, an array of news objects containing simple temporal data (`date`) mapping to specific `title` values such as "MATRADE Launches 22nd Edition of MIHAS".

## Component Integration
Injected into the `HomeView` media split-screen, and rendered extensively in full-page lists within the `MediaView`.
